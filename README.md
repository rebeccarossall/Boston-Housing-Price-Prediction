# Boston Housing Price Prediction

This project develops and evaluates a multiple linear regression model to predict median house prices in Boston neighbourhoods using R. The analysis includes exploratory data analysis (EDA), feature engineering, model selection using AIC and BIC, regression diagnostics, and validation on unseen test data. The final model is also benchmarked against a simple linear regression model to evaluate its predictive performance.

This project is based on a final-year university coursework assignment that achieved a mark of **91%**.

## Skills Demonstrated

- Exploratory Data Analysis (EDA)
- Multiple Linear Regression
- Feature Engineering
- Data Transformation
- Model Selection (AIC/BIC)
- Regression Diagnostics
- Predictive Modelling
- Data Visualisation
- Outlier & Influence Analysis
- R Programming
- Technical Report Writing

## Key Highlights

- Conducted exploratory data analysis (EDA) using scatterplots, correlation analysis, histograms, and summary statistics to investigate relationships between predictors and median house prices.
- Applied logarithmic transformations and feature engineering to improve model assumptions.
- Built and compared multiple regression models using predictor significance, adjusted R², AIC, and BIC.
- Assessed regression assumptions using residual analysis, QQ plots, and Cook's distance to identify influential observations.
- Generated and evaluated predictions on both training and independent test datasets by comparing predicted and observed values.
- Compared the final model against a simple linear regression baseline.
